CM Fibre — Clean Responsive Website

This repository contains a modern, mobile-first single-page website for CM Fibre.

Preview locally:

```bash
# from the project root
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

What's included:
- `index.html` — updated, accessible, mobile-friendly layout
- `style.css` — modern responsive styles
- `script.js` — small accessible menu toggle
